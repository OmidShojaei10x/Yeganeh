# گزارش ارزیابی کاندیدا - یگانه ولی‌محمدی

این repository شامل گزارش ارزیابی کاندیدا برای پوزیشن کمپین منیجر یکتانت است.

## راه‌اندازی GitHub Pages

برای نمایش لایو این فایل روی GitHub Pages:

1. این repository را روی GitHub ایجاد کنید:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git branch -M main
   git push -u origin main
   ```

2. در تنظیمات repository روی GitHub:
   - به Settings > Pages بروید
   - Source را روی "Deploy from a branch" تنظیم کنید
   - Branch را "main" و folder را "/ (root)" انتخاب کنید
   - Save کنید

3. بعد از چند دقیقه، فایل شما در آدرس زیر قابل دسترسی خواهد بود:
   ```
   https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/indes03.html
   ```

## فایل‌ها

- `indes03.html` - گزارش کامل ارزیابی کاندیدا
